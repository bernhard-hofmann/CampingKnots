About

Overhand Knot

Overview

Stopper knots are useful for putting a knot in a rope to stop it slipping through a hole that the rope has been passed through.

The overhand knot is the simplest stopper knot, but is not as good as the figure eight stopper because it typically becomes so tight under tension that it can't be undone. Though sometimes this is adventageous, so the knot is included here.





Figure Eight Knot

Overview

Stopper knots are useful for putting a knot in a rope to stop it slipping through a hole that the rope has been passed through.

The figure eight knot is useful as a stopper knot because it can more easily be undone by bending the knot over, even after a lot of tension has been put on it.





Square Knot

Overview

The square knot is useful for joining two lengths of rope that are of equal thickness.

One of the most common rhymes repeated when learning this knot is "left over right, then right over left".





Sheet Bend

Overview

This knot is useful for joining two ropes of unequal thickness. The tension on the knot presses the end of the thinner rope down to hold it in place.






Bowline Knot

Overview

This knot is particularly useful for tying around yourself because the loop will not pull tight under strain.

Another very useful feature of this knot is that it is easy to undo, by folding it open, even after it has been under significant tension.







Clove Hitch

Overview

This knot is useful for attaching a rope to a post or pole. The tension on the rope presses down on itself to increase grip as tension increases.






Evenk-style Knot

Overview

This knot is similar to the evenk knot. It is useful for attaching a line to a an object that can be constricted by the knot.





For added security, the working end can be passed through the open bite to prevent it coming undone.


To undo the knot, simply remove the working end from the bite and pull it hard to release the knot.

Prusik Knot

Overview

The prusik knot is very useful because it allows the knot to be moved up and down the line to set tension and will hold that tension when you let it go.









Alpine Butterfly Knot

The alpine butterfly knot it very useful for making loops in a line that needs to be tight. The loops can take tension as well, so a line with a few butterfly knots makes an ideal "rack" for hanging things on. It can also be used to join two lengths of equal thickness chord.
